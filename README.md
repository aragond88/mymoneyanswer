# MyMoneyAnswer

MyMoneyAnswer is a static personal finance education website for users in the United States and Canada. It helps beginners decide their next best money move in 5 minutes through simple tools, plain-English lessons, calculators, and curated finance resources.

All finance content on MyMoneyAnswer is educational information only. The site must not be presented as personalized financial, tax, legal, or investment advice.

## Current Pages

- `index.html`: Main quiz tool.
- `calculator.html`: Investment calculator.
- `directory.html`: YouTube channel directory.
- `blog.html`: Blog hub.

## Tech Stack

- Plain HTML.
- Plain CSS.
- Plain JavaScript.
- No framework.
- No backend.
- No database.
- No build step.
- Hosted on Vercel.

## Project Goals

- Make MyMoneyAnswer feel like a financial decision tool, not a generic finance blog.
- Help beginners in the United States and Canada understand practical next steps.
- Build a trustworthy content foundation before scaling Search Engine Optimization pages.
- Add calculators, money lessons, trust pages, and monetization carefully.
- Keep the site lightweight, accessible, fast, and easy to maintain.

## Important Rules

- Public website copy must be in English.
- Use relative paths only, for example `./calculator.html`, never `/calculator.html`.
- Do not make guaranteed income claims.
- Do not promise investment returns.
- Do not say "financial advice"; use "educational information" or "general information."
- Add disclaimers when content discusses investing, debt, taxes, affiliate products, or financial decisions.
- Do not add frameworks, advertising scripts, analytics scripts, or affiliate scripts without approval.
- Do not redesign existing public pages unless explicitly requested.
- Before editing, inspect current files because the user may have already made updates.
