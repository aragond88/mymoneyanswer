# Project Context

## Objective

MyMoneyAnswer helps beginners decide their next best money move in 5 minutes. The site should become a simple, trustworthy financial decision tool for users in the United States and Canada.

## Primary Markets

- United States.
- Canada.

## Public Language

Public website copy must be in English.

## Positioning

MyMoneyAnswer helps beginners decide their next best money move in 5 minutes. The experience should feel practical, calm, and tool-like rather than like a generic finance blog.

## Current Site Structure

- `index.html`: Main quiz tool.
- `calculator.html`: Investment calculator.
- `directory.html`: YouTube channel directory.
- `blog.html`: Blog hub.

## Monetization Strategy

Potential monetization methods include display ads, affiliate links, finance books, and finance platform comparison blocks. Monetization must be transparent and useful. Affiliate disclosures must appear before or near affiliate links. Product rankings must not be based only on commission.

Do not add advertising scripts, analytics scripts, or affiliate scripts without approval.

## Search Strategy

Search Engine Optimization should focus on helpful, beginner-friendly pages with clear titles, descriptions, internal links, and useful content. Each indexable page should have a unique purpose and should answer a real user question.

## Programmatic Search Engine Optimization Strategy

Programmatic Search Engine Optimization pages should be scaled carefully. Start with a small set of test pages, make sure each page has unique and useful content, review performance in Google Search Console, and scale only the page types that show traction.

Do not create thin pages that only swap numbers, locations, or keywords.

## Geo-targeting Strategy

The site supports users in the United States and Canada. Country detection may be used only to personalize the experience. Important content must not be hidden from search engines.

Always provide a manual country toggle. If country detection fails, default to United States and store the user's choice in `localStorage`.

## Brand Rules

- Public copy must be simple, helpful, calm, beginner-friendly, trustworthy, and practical.
- Use educational disclaimers when content discusses investing, debt, taxes, affiliate products, or financial decisions.
- Do not make guaranteed income claims.
- Do not promise investment returns.
- Do not present content as personalized financial, tax, legal, or investment advice.
- Use "educational information" or "general information."
- Use Inter everywhere.
- Use the primary green `#16A34A`.
- Keep calls to action green `#16A34A` or white with a green border.
