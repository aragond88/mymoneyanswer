# Roadmap

## Phase 1: Foundation

Set up the trust, navigation, footer, sitemap, robots file, and project structure needed before scaling the site.

## Phase 2: Core Experience

Improve the existing core pages and add the main money lessons and tools pages.

## Phase 3: Calculators

Build practical calculators that help beginners make clear financial decisions, starting with emergency fund and debt payoff calculators.

## Phase 4: Content

Publish the first educational articles and improve the YouTube directory into a more useful editorial resource.

## Phase 5: Programmatic Search Engine Optimization

Create a small group of test pages, review search performance, and scale only page types with useful content and proven traction.

## Phase 6: Monetization

Add transparent monetization after the trust pages and content foundation are ready. Use disclosure blocks, neutral comparisons, relevant affiliate links, and display ads only when approved.
